# src/db/

## Purpose

Drizzle ORM client, schema definitions, and migration runners.

## Key files

- `src/db/client.ts` — `createDb(url)` returns `{ db, sql }` (Drizzle ORM client + raw `postgres` connection). Used in app and tests.
- `src/db/schema.ts` — All Drizzle table definitions, types, and relations. Source of truth for the schema.
- `src/db/migrations/` — SQL migrations generated by `npx drizzle-kit generate`.

## Dependencies

- Upstream: `drizzle-orm`, `postgres`.

## Tests

- Schema is exercised indirectly by every other module's integration test.
